# 빵긋 백 앤드 서버


## 역할

* Domain + Api serer   

## 에그리거트

* ### 회원/인증
  회원 정보, 인증관련 등
  
* ### 빵집/점주
  빵집의 기본정보, 운영정보, 상태, 휴게관련 등

* ### 이벤트 
  빵집의 이벤트를 다룬다.
  
* ### 메뉴
  메뉴그룹, 일반 메뉴, 마감 메뉴, 갓 구운빵, 신메뉴, 메뉴 상태 (품절) 등
  
* ### 주문
  주문 접수, 접수 내역, 접수 상태 등을 다룬다.

* ### 결재
  위약금, 결재 수단, 결재 내역, 결재관련 등을 다룬다.

* ### 쿠폰
  쿠폰 등을 다룬다.
  
* ### 정산
  위약금을 빵집 사장님들에게 정산 해주는 것을 다룬다.

* ### 통계
  클릭수나, 조회수, 주문금액, 등을 다룬다.   
  서비스 성과와 서비스 이용료를 다룬다.
  
* ### 자주묻는질문
  
* ### 공지사항
  
* ### 이용약관
  
* ### 알림
